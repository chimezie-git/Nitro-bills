# Nitro Bills

An app for Nitro bills.

*An upwork project*


### Commands
add native splash
```bash
dart run flutter_native_splash:create
```

generate hive box
```bash
dart run build_runner build --delete-conflicting-outputs 
```

generate launcher icon
```bash
dart run flutter_launcher_icons
```
